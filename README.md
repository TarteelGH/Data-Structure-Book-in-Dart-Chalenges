# Data-Structure-Book-in-Dart-Challenges

This repository contains solutions to the challenges from the Kodeco book:  
**“Data Structures & Algorithms in Dart”** – Chapters on **Stacks** and **Linked Lists**.

---

## 📚 Topics Covered

- Basic Data Structures
- Stack Operations
- Singly Linked List Operations

---

## 📖 Source Material

All topics and challenges were based on the following Kodeco book chapters:

- 📘 [Chapter 3: Basic Data Structures in Dart](https://www.kodeco.com/books/data-structures-algorithms-in-dart/v1.0/chapters/3-basic-data-structures-in-dart)  
- 📘 [Chapter 4: Stacks](https://www.kodeco.com/books/data-structures-algorithms-in-dart/v1.0/chapters/4-stacks)  
- 📘 [Chapter 5: Linked Lists](https://www.kodeco.com/books/data-structures-algorithms-in-dart/v1.0/chapters/5-linked-lists)
